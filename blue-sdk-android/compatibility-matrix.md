## 已知限制与厂商适配

### 华为 / HarmonyOS
- Android 6~11 需要开启"位置服务"才能扫描 BLE（系统限制）
- 建议在扫描前检查位置服务状态

### 小米 / MIUI
- 需要在"自启动管理"中允许 APP，否则后台 BLE 连接可能被终止
- MIUI 12+ 需要在"电量和性能"中关闭"省电优化"

### 通用限制
- 模拟器不支持 BLE 硬件，需使用真实设备测试
- BLE UUID 当前使用通用串口服务 UUID 占位，待硬件方确认后更新
